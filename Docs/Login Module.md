## Authentication Module API DOCS

>### Add User:-
>- **Endpoint:** http://127.0.0.1:8080/authentication/addUser
>- **Methord:** POST
>- **Status Code:**
> i. 	201 CREATED: On Success.
> **- Request Data:**
> > ```js
> >{
> >		"userName":"hunter76",
> >		"password":1234,
> >		"role":"Mangment",
> >		"result":20
> >}
> **- Response Data:**
> > ```js
> >{
> >		"id": 43,
> >		"userName": "hunter76",
> >		"password": "1234",
> >		"role": "Mangment",
> >		"result": 20
> >}
>

>###  User LogOut:-
>- **Endpoint:** http://127.0.0.1:8080/authentication/logout

> ii. 400 BAD REQUEST: If User not Logged in.
